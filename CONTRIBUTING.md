# Contributing to ResumeApex Eval

## Governance

ResumeApex Eval is governed under the [DGAF Framework](https://github.com/ndrorchestration/DGAF-Framework), meta-orchestrated by **Agent Amethyst**, with evaluation standards maintained by **Agent Apogee** (QA Orchestrator, L4).

## What This Repo Contains

- DGAF-certified 3-layer benchmark (Bronze / Silver / Gold)
- Dataset 1 evaluation cards
- Prompt quality scoring rubrics (15-scale: clarity, safety, reusability)
- NIST AI RMF-aligned audit infrastructure

**Note:** This repo contains evaluation architecture and benchmark tooling only. No personal data, no resume content.

## Attribution & Standards

- **NIST AI RMF** — Risk management alignment
- **DGAF 11Q Framework** — 11-question deployment gate
- **S-TIER Certification** — 97% confidence, zero-hallucination threshold
- **Paul-Elder Critical Thinking Framework** — scoring rubric foundation

## How to Contribute

1. Open an issue describing the benchmark gap or eval improvement
2. Fork and branch from `main`
3. All scoring changes must reference the relevant DGAF pattern
4. Submit PR with before/after benchmark results
